# 🌱 Nueva Esperanza – Sitio Web ONG

## 📌 Descripción del proyecto
Este proyecto consiste en el desarrollo de un sitio web para la ONG **Nueva Esperanza**, una organización dedicada a brindar apoyo a personas en situación de calle.

El objetivo principal de la página es visibilizar la problemática, dar a conocer el trabajo de la organización y presentar sus programas e iniciativas orientadas a la inclusión social y mejora de la calidad de vida.

El sitio está compuesto por las siguientes páginas:
- **Inicio**: presentación general de la ONG  
- **Nosotros**: historia, valores y equipo  
- **Programas**: iniciativas y acciones de la organización  
- **Contacto**: información para comunicarse
